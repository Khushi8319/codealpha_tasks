1: CGPA Calculator

Description
A C++ program that calculates semester GPA and overall CGPA based on course grades and credit hours across multiple semesters.

Features
- Accepts multiple semesters as input.
- Takes course name, grade, and credit hours for each course.
- Converts letter grades into grade points.
- Calculates semester-wise GPA.
- Calculates overall CGPA.
- Displays course details and final CGPA.

Concepts Used
- Functions
- Vectors
- Strings
- Loops
- GPA/CGPA Calculation Logic
